How it works

Two modes. One rule: build it in, don't bolt it on.

Create mode

Co-drafts the work you're already doing

You're writing a problem statement, a PRD, a user story. This mode drafts alongside you — asking the questions that surface exclusion while a sentence is being written, so the thinking shifts upstream.

Learn mode

Explore the field, or find what nobody's building

No task, no deadline — just exploring inclusive design as a discipline, or using a real constraint as a generative prompt to surface opportunities nobody's addressing yet.
